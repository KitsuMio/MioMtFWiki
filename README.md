# MioMtFWiki

> 来自社区、服务社区的现代跨性别知识库

MioMtFWiki 是一个社区驱动的跨性别知识库项目，致力于用清晰、准确、易于传播的方式整理跨性别相关知识。

我们发现，许多跨性别相关资料虽然内容详尽，却往往过于冗长、难以阅读，也难以回应社区中不断出现的新问题。与此同时，大量新人仍然依赖社交媒体、群聊和口耳相传获取信息，使误解、谣言与风险不断重复传播。

MioMtFWiki 希望改变这一现状。

## 我们的目标

我们的目标不是成为另一本医学教材，也不是成为一个脱离现实的资料档案馆。

我们的目标只有一个：

> 让正确答案比群聊里的错误信息传播得更快。

## 项目特点

### ✅ 结论优先

第一屏就是答案，而不是背景、定义、历史铺垫完才告诉你结论。

### 🌐 来自社区

内容来自社区正在问的真实问题，而不是维护者觉得重要的内容。

### 🔄 持续更新

社区出现新问题，就产生新条目，而不是几年更新一次的静态资料。

### 🧭 贴近当下

采用当前社区实际使用的概念与讨论框架，而不是停留在过去的叙事方式。

## 内容范围

目前收录内容包括：

- HRT 入门与常见方案
- 性别认同相关概念科普
- 常见误解与刻板印象澄清
- 现实生活经验与建议
- 社区高频问题整理

未来还将持续扩展：

- 家庭与出柜
- 学校与校园环境
- 隐私保护
- 社区踩坑案例
- 常见风险与误区

## 为什么做这个项目

跨性别相关资料并不少。

真正缺少的是：

- 能在几分钟内看完的资料
- 看完就知道下一步做什么的资料
- 愿意被转发给朋友的资料
- 能回应当下社区问题的资料

MioMtFWiki 希望成为这条缺失的中间道路。
