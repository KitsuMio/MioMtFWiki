# 简单明确的用药方案

::: warning 在继续之前

这篇文章是给「不想系统学习 HRT，只想先知道常见方案」的人准备的。

它会尽可能把内容压缩到最少，但代价是省略了大量原理、个体差异和剂量调整逻辑。

如果你有时间，仍然建议先阅读 [HRT入门](./hrt-start)。知道自己为什么这样用药，永远比记住一个方案更重要。

:::

::: tip 关于抗雄

如果抗雄压制不理想，不要盲目增加抗雄药剂量，优先增加雌激素剂量

:::

### 方案 A

这是本站最推荐的组合。雌二醇凝胶能够提供较稳定的雌激素水平，更容易在不用大剂量抗雄药的情况下降低睾酮水平，同时避开口服雌激素的首过代谢。

- **药物组合**：醋酸环丙孕酮（色普龙） + 雌二醇凝胶 (爱斯妥/Estrogel)
- **用法用量**：
    - **CPA**：每天一次，`10 ~ 12.5 mg`（口服）。
    - **雌二醇凝胶**：每天 `1 ~ 2 计量尺` 或 `2 ~ 4 泵`（涂抹于大腿内侧或手臂）。
- **副作用**：色普龙可能导致抑郁和情绪波动。

### 方案 B

目前最常见、最成熟的经典方案。药物获取方便、使用简单、抗雄效果强，但口服雌激素存在首过代谢，可能导致压不住雄激素而去超量服用色普龙，而超量的色普龙可能带来情绪相关副作用。

- **药物组合**：醋酸环丙孕酮 (色普龙) + 雌二醇片 (补佳乐)
- **用法用量**：
    - **CPA**：每天一次，`10 ~ 12.5 mg`（口服）。
    - **补佳乐**：每天 `2 ~ 6 mg`（分两次**舌下含服**）。
- **副作用**：色普龙可能导致抑郁和情绪波动，口服雌二醇可能有血栓风险。

### 方案 C

这种组合避开了色普龙的肝脏风险和口服雌激素的首过代谢，是目前最温和的组合。

- **药物组合**：螺内酯 (安体舒通) + 雌二醇凝胶 (爱斯妥/Estrogel)
- **用法用量**：
    - **螺内酯**：每天 `100 ~ 400 mg`（分两次口服）。
    - **雌二醇凝胶**：每天 `1 ~ 2 计量尺` 或 `2 ~ 4 泵`（取决于购买的包装类型）（涂抹于皮肤）。
- **副作用**：螺内酯有利尿作用和电解质紊乱问题。

### 方案 D

适合对 CPA 产生严重副作用（如极度抑郁、肝功能波动）或希望用药过程相对温和的人。

- **药物组合**：螺内酯 (安体舒通) + 雌二醇片 (补佳乐)
- **用法用量**：
    - **螺内酯**：每天 `100 ~ 400 mg`（通常分两次口服）。
    - **补佳乐**：每天 `2 ~ 6 mg`（分两次**舌下含服**）。
- **副作用**：螺内酯有利尿作用和电解质紊乱问题，口服雌二醇可能有血栓风险。
