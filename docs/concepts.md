# 概念与术语

## 性别相关基础概念

::: info 性别认同
一个人内心确认自己是什么性别。与出生时被指派的性别可能相同（顺性别），也可能不同（跨性别）。不取决于外表、穿着或喜欢谁。
:::

::: info 性别表达
一个人通过穿着、发型、言行举止展现出来的性别风格。可以是女性化、男性化或中性化，和性别认同是两件独立的事——表达女性化不代表认同为女性。
:::

::: info 性取向
一个人喜欢什么性别的人，与自己的性别认同无关。跨性别者的性取向同样多样，不因为是跨性别就限定喜欢某一性别。
:::

::: info 顺性别（Cisgender）
性别认同与出生时被指派性别一致的人。
:::

::: info 跨性别（Transgender）
性别认同与出生时被指派性别不一致的人。是一个总称，下面再细分方向和群体。
:::

::: info GD（Gender Dysphoria）
性别不安/性别焦虑。指因为身体特征、社会角色与自己性别认同不一致而产生的心理痛苦。是医学诊断术语，不是"矫情"或"想多了"。
:::

## 跨性别方向分类

::: info MTF / 跨女
出生时被指派为男性、性别认同为女性的跨性别者。可能会进行 HRT 或 SRS，也可能不进行，这不影响身份本身成立。
:::

::: info FTM / 跨男
出生时被指派为女性、性别认同为男性的跨性别者。
:::

::: info Non-binary（非二元）
性别认同不完全落在"男"或"女"二元框架内的人，可能认为自己是两者都是、都不是，或介于之间。
:::

## 容易混淆但不是跨性别的身份

::: info CD（Crossdresser，异装者）
穿着另一性别服装的人，性别认同与出生指派性别一致，不寻求长期的身份或身体改变。穿衣是表达或爱好，不是性别认同问题。
:::

::: info 男娘 / 伪娘

通常指性别认同为男性或非二元（MtX）、但穿女装/扮演女性化形象的人，本质是一种表达或角色扮演，跟 MtF(跨性别女性) 没有关系。

此类群体可能服用雌激素以追求效果，但这和 MtF 的概念是无关的。

:::

::: info 这两者的共同点
性别认同不是女性，变的是表达。如果一个人说"我认为自己本质上是女性"，而不只是"我喜欢女装"，那就不是这一类，是 MTF。
:::

## HRT 相关 {#hrt-related}

::: info HRT（Hormone Replacement Therapy）

性别肯定激素治疗。通过药物调整体内激素水平，使身体特征更接近自己认同的性别。MTF 通常需要抗雄激素 + 雌激素两类药物配合。

是身体层面的性别肯定医疗手段之一，不是跨性别身份成立的必要条件——是否 HRT 不影响一个人是不是跨性别。

:::

::: info 药娘 / yn / xyn

社区俗称，通常指正在或已经进行 HRT 的 MTF，但偶尔也会用来指代服用雌激素的 MtX 或男娘。

总之，只要某人的指派性别不为女，但服用雌激素，就可以被称为药娘。

所以，这不是一个能完整体现跨性别女性身份的词语。

MtF 不都是药娘，药娘不一定是 MtF。

:::

::: info 木桶饭
MTF 的谐音替代说法，源自拼音联想，纯粹是网络黑话，用于规避平台审核或图省事，没有额外含义。
:::

::: info 糖 / 吃糖
"糖"泛指 HRT 药物，"吃糖"指服用 HRT 药物进行治疗。是社区里最常见的隐语，便于在不方便直说的场合交流。
:::

::: info 补佳乐 / 补子 / 补
补佳乐是一种口服雌二醇药物，常用于 HRT。"补子""补"是社区对这种药的简称。
:::

::: info 爱斯妥
雌二醇凝胶，透皮吸收的雌激素制剂，避开口服药的肝脏首过代谢。
:::

::: info 色普龙
醋酸环丙孕酮（CPA），抗雄激素药物，HRT 中用于压低睾酮水平。
:::

## 医疗/法律文件

::: info 小证
写有相关诊断的病历或诊断书，通常用于支持开始 HRT。要求和获取方式因地区和医院而异。
:::

::: info 大证
正式的性别认同障碍/性别不安诊断证明，效力高于小证，通常是申请 SRS 等手术的前提条件之一。
:::

::: info SRS（Sex Reassignment Surgery）

性别重置手术，俗称变性手术。

是身体层面的性别肯定医疗手段之一，不是跨性别身份成立的必要条件——做不做手术不影响一个人是不是跨性别。

:::

## pass 与身份处境

::: info pass
指一个人的性别表达在他人（尤其陌生人）视角下，被自然识别为自己认同的性别，而不被识别出跨性别身份。是表达层面的结果，不是身份是否"真实"的判断标准。
:::

::: info 鉴跨
依据嗓音、外貌等特征，对一个人是否 pass、是否符合性别刻板印象做出主观评判。这种行为通常带有冒犯性，在社区内不受欢迎，因为它把"是否符合外界期待"和"身份是否成立"混为一谈。
:::

::: info 出柜
主动、自主地公开自己的跨性别身份。
:::

::: info 爆柜
身份在未做好准备或未主动选择的情况下被他人发现，被迫公开。和"出柜"的核心区别在于主动权——爆柜往往伴随更高的风险和更少的准备时间。
:::

## 家庭与处境分类（社区俗称）

::: info 家长党
取得家长理解和支持的跨性别者。
:::

::: info 家暴党
因跨性别身份在家庭中遭受暴力对待的跨性别者。
:::

::: info 年龄党
在青春期较早阶段就开始 HRT 的跨性别者，通常意味着身体可塑性更高。
:::

::: info 天赋党
先天身体条件（骨骼、声音、体脂分布等）较接近自己认同性别的跨性别者，HRT 后女性化效果通常更明显。
:::

::: info 天残党
与"天赋党"相对，先天身体条件与认同性别差距较大，HRT 效果受限更多，需要更多其他手段（手术、声音训练等）辅助。
:::

## 社会立场分类

::: info 友跨
支持跨性别群体、尊重其性别认同和权利的立场。
:::

::: info 反跨
否定或排斥跨性别者性别认同、性别表达或身份本身的立场。
:::
